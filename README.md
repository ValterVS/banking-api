# BANKING-API

Olá sou Valter da Silva, sou graduando de Engenharia de Software no 10° semestre, e esse projeto tem como objetivo reunir as principais stacks que tenho conhecimento, e fazer uma aplicação útil, real e muito utilizada no dia a dia dos bancos.

A Banking API é uma aplicação backend desenvolvida para simular as operações essenciais de um banco digital.

O projeto foi criado com o objetivo de evoluir continuamente até se tornar uma API pronta para produção, incorporando boas práticas de desenvolvimento backend utilizadas por empresas internacionais.

## Funcionalidades planejadas
- Cadastro de clientes
- Criação de contas
- Depósitos
- Saques
- Transferências
- Histórico de transações
- Autenticação com JWT
- Integração com PostgreSQL
- Documentação com Swagger
- Containerização com Docker
- Deploy em AWS
- Testes automatizados
- CI/CD com GitHub Actions

## Tecnologias (roadmap)
- Java 21
- Spring Boot
- Spring Security
- PostgreSQL
- Docker
- AWS
- GitHub Actions
- JUnit
- Maven
